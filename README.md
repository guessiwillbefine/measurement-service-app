### about
This project is an example of a sensor monitoring system using web technologies and interface. Current stack of technologies we used:
- Java 17
- Spring boot 3.0.1
  - Hibernate
  - Spring Data JPA
  - Security (JWT)
  - WebSockets
- RabbitMQ
- MySQL, H2 (with liquibase as migration tool)
- Docker
- Swagger for docs
